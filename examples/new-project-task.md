# Work type

NEW_PROJECT

# Outcome

Create a standalone Rust + egui YouTube Project Manager.

# Scope

Milestone 1 only:

- initialize project structure;
- create and open a video project folder;
- display a project list;
- persist basic project metadata;
- create `spec.md`, `plan.md`, `todos.md`, `test.md`, and `final.md`.

# Out of scope

- video editing;
- cloud synchronization;
- publishing to YouTube;
- installer and release packaging.

# Acceptance criteria

- application builds;
- application launches;
- a project can be created;
- metadata can be saved and reopened;
- a smoke test or equivalent verification exists;
- project control files reflect milestone status.

# Memory rule

Search relevant project memory before implementation and write back only verified reusable lessons.

# Stop

Stop before destructive filesystem operations, deployment, commit, push, secrets, or scope expansion.
