# New Project Pipeline

## Required artifacts

README.md, AGENTS.md, spec.md, plan.md, todos.md, test.md, final.md, .gitignore.

## Stages

1. Codex creates the project contract.
2. Scaffold agent creates a minimal runnable skeleton.
3. Builder implements the first vertical slice.
4. Tester runs smoke and unit tests.
5. Reviewer checks architecture, security, and maintainability.
6. Codex verifies from a clean command sequence.
