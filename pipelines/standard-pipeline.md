# Standard Pipeline

1. Preflight
2. Baseline git status
3. Planner analysis
4. Codex approves executable plan
5. Builder implementation
6. Tester verification
7. Reviewer diff review
8. Codex independent verification
9. Update todos/test/final
10. Report evidence

## Stop conditions

- Destructive operation requested
- Scope expansion required
- Credentials required
- Three repair loops failed
- Unrelated existing changes make attribution unsafe
